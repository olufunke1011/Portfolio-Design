    Expertise

    UI/UX Design

    The “UI” in UI design stands for “user interface.” The user interface is the graphical layout of an application. The UI designer decides what the application is going to look like. “UX” stands for “user experience.” A user’s experience of the app is determined by how they interact with it.

    FRONT END

    A front-end developer links together the world of design and technology, packaging up the utility of the back end in an inviting way for users to interact with. They take website design files and convert them into HTML, JavaScript (JS) and/or CSS code - the core elements of front-end development. This includes design/layout, content, buttons, images, navigation, and internal links.

    BACK END

    Back-end Development refers to the server-side development. Backend developer focuses on databases, scripting, and the architecture of websites. Code written by back-end developers helps to communicate the database information to the browser.
